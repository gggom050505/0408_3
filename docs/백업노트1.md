# 백업노트1 (재현 런북)

이 문서는 **현재 저장소 상태를 다시 재현**하기 위한 체크리스트입니다.  
기획 배경은 `docs/MAKING_NOTES.md`를 참고하고, 이 문서는 실행/검증 순서만 다룹니다.

최종 정리: **2026-04-29**

## 2) 핵심 기능 재현 순서

### STEP 1. 로그인/홈 진입
1. 앱 실행 후 로그인(신규/기존/로컬 계정).
2. 홈 화면 및 GNB 탭 노출 확인.

### STEP 2. 가방 · 「한국전통 메이저카드」 덱 선택 시 무엇이 바뀌나

**언제 목록에 보이나**
- 보유 아이템에 `korea_major_card`(한국전통 메이저 **조각**)가 **1장 이상** 있으면, 가방의 카드 덱 목록에 **「한국전통 메이저카드」** 행이 자동으로 추가됩니다.  
  (상점에서 덱 품목 `korea-traditional-major`를 카드 타입으로 이미 샀다면, 동일 ID로 첫 번째 루프에서도 잡힙니다.)

**선택(장착) 시 저장**
- `로컬 프로필`의 `equipped_card`가 **`korea-traditional-major`** 로 저장됩니다(`LocalShopRepository.equip`).

**타로 탭에 반영**
- 홈이 프로필을 읽어 `TarotTab`에 `equippedCardThemeId`·**보유 한국전통 메이저 번호 목록**(`ownedKoreaMajorCardIds`, 조각의 타로 id 0~21)을 넘깁니다.
- **드로우 풀**: 클레이 마이너 60장에 해당하는 카드 + 메이저 0~21 — 한 덱으로 섞은 뒤 **22장만** 뽑아 9슬롯 스프레드에 사용합니다(`buildMinorClayAndKoreaTraditionalFullDrawPool`, `_deckN == 22`).
- **앞면 일러스트**
  - **마이너**: 클레이 마이너 일러(`major-clay` 계열 경로).
  - **메이저(0~21)**: 그 번호 조각을 **가방에 보유했으면** 한국전통 에셋, **없으면** 클레이 메이저로 **자동 보완**(`resolveFrontThemeForKoreaTraditionalDeckCard`).
- 덱을 바꾸거나 보유 조각 목록이 바뀌면 세션을 비우고 다시 섞을 수 있음(`TarotTab.didUpdateWidget`).

**혼합 덱과 구분**
- **「마이너 + 한국전통 메이저 (혼합)」**(`mixed-minor-korea-major`)은 마이너 전체 + **보유 중인 한국전통 메이저 번호만** 메이저로 넣는 별도 풀입니다. 위 「한국전통 메이저카드」와 목적이 다릅니다.

**검증 포인트**
1. 조각 1장 이상 확보 후 가방 → 카드 덱에서 「한국전통 메이저카드」 선택 → 스낵바·`(장착)` 표시.
2. 타로 탭에서 메이저가 나왔을 때, 보유 번호는 한국전통 그림·미보유 번호는 클레이 메이저 그림인지 확인.

관련 파일:
- `lib/widgets/bag_tab.dart` — 덱 목록·안내 문구 `_koreaTraditionalDeckEquipHint`
- `lib/standalone/local_shop_repository.dart` — `equipped_card` 저장
- `lib/widgets/home_screen.dart` — `ownedKoreaMajorCardIds` 계산 후 `TarotTab`에 전달
- `lib/widgets/tarot_tab.dart` — 풀 구성·앞면 테마 분기
- `lib/data/deck_card_catalog.dart` — `buildMinorClayAndKoreaTraditionalFullDrawPool`, `resolveFrontThemeForKoreaTraditionalDeckCard`
- `lib/data/card_themes.dart` — `koreaTraditionalMajorThemeId`
- `lib/config/korea_major_card_catalog.dart` — 조각 상품 ID ↔ 타로 id
- `lib/tarot/ai_reading_prompt.dart` — AI 리딩 시 장착 덱 톤(`equippedCardThemeId`) 반영

### STEP 3. 오프닝 스플래시
1. 앱 시작 시 `assets/opening/` 이미지 캐러셀 노출 확인.
2. 점 인디케이터가 이미지 개수와 맞게 표시되는지 확인.
3. 터치 시 다음 단계로 넘어가는지 확인.

관련 파일:
- `lib/config/opening_assets.dart`
- `lib/widgets/splash_screen.dart`
- `assets/opening/open(1).png` ~ `open(3).png`

### STEP 4. 간지 달력
1. 간지 달력 탭에서 날짜 이동/선택 확인.
2. 상단 **오행 안내 배너** 펼침 확인.
3. 날짜 셀 탭 시 바텀시트가 열리고 일진/연간/월간 패턴이 보이는지 확인.

관련 파일:
- `lib/widgets/ganji_calendar_tab.dart`
- `lib/data/ganji_sixty_day_patterns.dart`
- `lib/data/five_elements_guide.dart`

### STEP 5. 타로 탭
1. 카드 설명 보기 스위치를 켜기/끄기.
2. 앱 재실행 후 같은 값으로 유지되는지 확인.

관련 파일:
- `lib/widgets/tarot_tab.dart`
- `lib/standalone/local_app_preferences.dart`

### STEP 6. 오늘의 타로
1. 10칸 슬롯 채우기.
2. **한 번에 뒤집기** 버튼 노출/동작 확인.
3. 카드 설명 보기 스위치 켜기/끄기 후 재실행 유지 확인.
4. 완료 후 결과/게시 흐름 정상 확인.

관련 파일:
- `lib/widgets/today_tarot_screen.dart`
- `lib/standalone/local_app_preferences.dart`
